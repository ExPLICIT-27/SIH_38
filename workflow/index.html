System Architecture

Clients

  • Flutter app (Android/iOS) - offline-first forms, media, GPS
  • Web dashboard (future) - verification, analytics

APIs

  • NestJS REST - auth, roles, uploads, projects, registry
  • JWT auth - role-based access control (guards and decorators)

Data & Storage

  • PostgreSQL (Prisma) - normalized project, org, upload, registry tables
  • IPFS - content-addressed artifacts, tamper evidence

Smart Contracts

  • ERC-1155 credits - batchable tokens for credit vintages
  • Registry contract - provenance and state transitions

Infra

  • Docker Compose - API, DB, IPFS gateway
  • CI friendly - tests, migrations, typegen

Security

  • JWT + RBAC - reviewers vs submitters
  • Content hashes - verify media integrity
